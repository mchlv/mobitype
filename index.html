A Designers Guide to Mobile Typography

With our modern "mobile first" mentality, typography now requires additional consideration.

Choosing a font

Each typeface is unique, and needs to be chosen with care. Typography is more than just pretty letters on a screen - it is an arrangement of words to be simultaneously readable, legible, and interesting.

Font Types

Serif

Sans-Serif

This is a serif font

Serif fonts have a small stroke at the end of a letters main stroke. Traditionally used in print, this font style is often used on screen for headings, table text, or captions. A serif font is best used to convey a strong, professional design style.

This is a sans-serif font

Sans Serif fonts do not have the extended stroke that Serif fonts have. This font style is most commonly used for body text - as it's clean look allows for improved readability. It is also quite versatile - it can be used for headings, body text, captions, and more!

Visual Hierarchy

Hierarchy is to emphasize the key elements and create contrast. Mobile screens have a more limited space, so many designers apply only 2 levels. A headline and a body will suffice; headline is to grab the reader instantly and the body toward readability.

Subheadings

Leading

Leading is the space between lines. Mobile screen is smaller, so leading is usually smaller than a desktop version. Small fonts need more spacing. Line spacing as a percentage should actually increase with smaller font sizes.

CTA

A "Call to Action" is the desired point of interaction with your user. Short, glanceable strings of text lead to faster reading and greater comprehension when the font is large, all caps, and not condensed.

White Space & Margins

This is a heading

This is body text. For best practice, it is recommended to use sans-serif fonts as body text, especially on mobile. The smooth, curvy letters make it much easier to read. Also consider the font size - 16px is the recommended minimum on mobile.

Keep it clean

UI = User Interface

An interface is the screen that your users interact with. Since mobile view has limited space for content, we must prioritize readablilty and visual hierarchy above all else. This elevates the user experience and ensures the information will be easily processed and consumed by the user.

All about the user

UX = User Experience

The user experience is the most crucial factor in a successful mobile design. When designing for a smaller screen, we need to consider the readability of the text, for users of all ages. A font should be big enough for users to read easily but not too big to confuse typographic hierarchy.

Improving UX

  • Use dark grey as your primary font colour - black can be straining on the eyes
  • Simple fonts, clean style, and a minimum of elements is recommended.
  • Avoid decorative and cursive fonts.
  • Paragraph line length should be between 30-40 characters per line.
